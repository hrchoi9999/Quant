# Redbot Model Copy Guide

## 목적
이 문서는 Redbot 서비스 화면, 리포트, 소개 문구에서 사용할 기본 카피 기준을 정리한다.

## 안정형
- 기본 문구:
  - 하락장에 대비해 채권, 달러, 금 같은 방어 자산 비중을 높인 보수형 포트폴리오입니다.
- 보조 문구:
  - 수익 극대화보다 낙폭 방어와 안정성을 우선합니다.

## 균형형
- 기본 문구:
  - 주식과 ETF를 함께 사용해 수익과 안정성의 균형을 추구하는 포트폴리오입니다.
- 보조 문구:
  - 과도한 변동성은 피하면서 중장기 성장을 목표로 합니다.

## 성장형
- 기본 문구:
  - 상승 추세에서 더 높은 수익을 추구하는 공격형 포트폴리오입니다.
- 보조 문구:
  - 변동성은 더 크지만 성장 국면에서 적극적으로 대응합니다.

## 자동전환형
- 기본 문구:
  - 시장 국면에 따라 포트폴리오 구성이 자동으로 바뀌는 동적 포트폴리오입니다.
- 보조 문구:
  - 상승, 보합, 하락 환경에 맞춰 내부 전략이 자동 전환됩니다.

## 카피 작성 원칙
- 사용자에게는 기술 용어보다 결과와 성향 중심으로 설명한다.
- ADX, RSI, ATR 같은 내부 지표명은 사용자 문구에서 제외한다.
- “안정형 / 균형형 / 성장형 / 자동전환형” 네 가지 이름을 일관되게 유지한다.
- 내부 모델명 `S2`, `S3`, `S4`, `S5`, `S6`, `Router`는 운영 문서와 관리자 리포트에서만 사용한다.
